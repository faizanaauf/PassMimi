# Contributing to PassMimi

Thank you for your interest in contributing to PassMimi.

PassMimi is an open-source password security analyzer focused on privacy, usability, and real-world password security. Every contribution helps improve the project for developers and users worldwide.

---

# Ways to Contribute

There are many ways you can help.

## Report Bugs

If you discover a bug:

- Search existing Issues first.
- Create a new Issue if one does not already exist.
- Include clear steps to reproduce the problem.
- Include screenshots if applicable.
- Mention your operating system and Python version.

---

## Suggest Features

Feature requests are welcome.

Please describe:

- The problem you are trying to solve.
- Your proposed solution.
- Alternative approaches you considered.

Clear feature requests help guide future development.

---

## Improve Documentation

Documentation improvements are always appreciated.

Examples include:

- Fixing typos
- Improving explanations
- Adding examples
- Creating tutorials
- Improving installation instructions

---

## Submit Code

Before submitting code:

1. Fork the repository.
2. Create a new branch.

```bash
git checkout -b feature/your-feature-name
```

3. Make your changes.

4. Commit your work.

```bash
git commit -m "Add your feature"
```

5. Push to your fork.

```bash
git push origin feature/your-feature-name
```

6. Open a Pull Request.

---

# Coding Guidelines

Please follow these guidelines:

- Write clean, readable code.
- Keep functions small and focused.
- Add comments where necessary.
- Follow existing project structure.
- Test your changes before submitting.

---

# Pull Request Checklist

Before opening a Pull Request, make sure:

- Code builds successfully.
- Existing functionality still works.
- New functionality is tested.
- Documentation is updated if needed.
- No unnecessary files are included.

---

# Development Goals

PassMimi aims to become a complete open-source password security platform.

Areas where contributions are especially valuable include:

- Password security research
- Performance optimization
- Cross-platform compatibility
- UI improvements
- Accessibility
- Documentation
- Testing
- Enterprise security features

---

# Community

Please be respectful and constructive when interacting with other contributors.

We welcome developers of all experience levels.

Thank you for helping improve PassMimi.

⭐ Every contribution, no matter how small, is appreciated.
